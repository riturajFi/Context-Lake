# Architecture Notes

This repository currently includes the Milestone 1 monorepo skeleton plus the Milestone 2 contract and persistence foundation.

- `services/` contains independently bootable APIs and workers.
- `packages/` contains shared runtime concerns such as config, logging, HTTP bootstrap, and common types.
- `infra/` contains local development infrastructure support files.
- `docs/` contains milestone-level documentation and implementation notes.

For Milestone 2 data contracts and schema details, see:

- `docs/data-model.md`
- `docs/event-catalog.md`
